# Credits

JuicyDocs is built with amazing open-source tools and contributions from the community.

## Core Technologies

### Eleventy

[Eleventy](https://www.11ty.dev/) is the static site generator that powers JuicyDocs. It's simple, fast, and flexible.

### Tailwind CSS

[Tailwind CSS](https://tailwindcss.com/) provides the utility-first CSS framework used for all styling in JuicyDocs.

### Pagefind

[Pagefind](https://pagefind.app/) powers the lightning-fast search functionality.

### Shiki

[Shiki](https://shiki.style/) provides beautiful syntax highlighting using VS Code's grammar files.

## Icons & Fonts

### Phosphor Icons

[Phosphor Icons](https://phosphoricons.com/) provides the beautiful, flexible icon set used throughout JuicyDocs.

### General Sans

[General Sans](https://www.fontshare.com/fonts/general-sans) is the primary typeface, provided by Fontshare.
