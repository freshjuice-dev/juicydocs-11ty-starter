We're almost at 1.0! This release adds search functionality, table of contents, and lots of polish.

## What's New

### Command Palette Search

- Integrated Pagefind for lightning-fast search
- Keyboard shortcut (press `S` or `/`)
- Fuzzy matching support
- Search result previews

### Table of Contents

- Auto-generated from page headings
- Scroll spy highlights current section
- Desktop sidebar and mobile drawer
- Smooth scroll to sections

### Navigation Enhancements

- Previous/Next page links
- Improved breadcrumb styling
- Active state indicators
- Nested navigation support

### Accessibility

- Added skip-to-content link
- Improved keyboard navigation
- Better focus indicators
- ARIA labels throughout

### Performance

- HTML minification in production
- Optimized asset loading
- Speculation rules for instant navigation

## Bug Fixes

- Fixed TOC not updating on route change
- Corrected search index generation
- Fixed mobile menu z-index issues
- Resolved theme switcher accessibility

## What's Left for 1.0

- Final documentation review
- Performance audit
- Cross-browser testing
