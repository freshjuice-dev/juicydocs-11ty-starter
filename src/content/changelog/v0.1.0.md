The very first release of JuicyDocs - just a raw idea turned into code.

## What's Included

### Initial Setup

- Basic Eleventy 3.x configuration
- Simple page layout with header and footer
- Markdown content support
- Basic CSS styling

### Early Decisions

- Chose Nunjucks as the templating engine
- Set up the project structure
- Created the first documentation page

## Known Limitations

- No navigation system yet
- No dark mode
- No search functionality
- Mobile experience needs work

This is just the beginning. Stay tuned for more updates!
