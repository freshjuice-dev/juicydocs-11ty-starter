JuicyDocs is starting to take shape! This release adds the foundational navigation system.

## What's New

### Navigation

- Added sidebar navigation component
- Implemented `eleventyNavigation` for automatic menu generation
- Basic breadcrumb support

### Styling

- Introduced Tailwind CSS for utility-first styling
- Created base typography styles
- Added responsive breakpoints

### Developer Experience

- Split configuration into modular files
- Added custom filters for content processing
- Improved build performance

## Bug Fixes

- Fixed heading hierarchy issues
- Corrected link styling in content areas

## Up Next

We're working on dark mode and search functionality for the next release.
