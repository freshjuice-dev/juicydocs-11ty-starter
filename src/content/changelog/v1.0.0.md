We're excited to announce the first stable release of JuicyDocs!

## What's New

### Core Features

- **Eleventy 3.x** - Built on the latest version of Eleventy for fast builds
- **Tailwind CSS v4** - Modern utility-first CSS with the new v4 engine
- **Dark Mode** - System-aware theme switching with manual override
- **Command Palette Search** - Fast, keyboard-navigable search powered by Pagefind

### Navigation

- Sidebar navigation with nested pages
- Auto-generated table of contents with scroll tracking
- Breadcrumb navigation
- Previous/Next page navigation

### Developer Experience

- YAML configuration files for easy setup
- Modular Eleventy config structure
- Shiki syntax highlighting with all languages
- Phosphor icons integration

### Mobile Experience

- Responsive design that works on all devices
- Bottom navigation bar for quick access
- Slide-out sidebar menu
- Touch-friendly TOC drawer

## Getting Started

Check out the [Getting Started](/docs/) guide to begin using JuicyDocs for your documentation.
