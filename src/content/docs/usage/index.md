Welcome to JuicyDocs! This section covers everything you need to get started.

## Quick Start

```bash
# Clone the repository
git clone https://github.com/freshjuice-dev/juicydocs-11ty-starter.git my-docs
cd my-docs

# Install dependencies
npm install

# Start development server
npm run dev
```

Your documentation site will be available at `http://localhost:8080`.

## What's Included

JuicyDocs provides a complete documentation solution:

- **Modern Stack** - Built with Eleventy 3 and Tailwind CSS v4
- **Fast Search** - Command palette search powered by Pagefind
- **Dark Mode** - System, light, and dark theme options
- **Mobile Ready** - Responsive design with touch-friendly navigation
- **SEO Optimized** - Sitemap, RSS feed, and meta tags included
