This section contains live examples demonstrating various JuicyDocs features.

## Available Examples

- **[Shortcodes](/docs/examples/shortcodes/)** - Alerts, embeds (YouTube), icon cards
- **[Buttons](/docs/examples/buttons/)** - Button styles, variants, and sizes
- **[Typography](/docs/examples/typography/)** - Headings, paragraphs, lists, and text formatting
- **[Code Blocks](/docs/examples/code/)** - Syntax highlighting examples in various languages
- **[Table of Contents](/docs/examples/toc/)** - Page demonstrating TOC behavior
- **[404 Page](/docs/examples/404/)** - Custom error page

## Purpose

These examples serve two purposes:

1. **Documentation** - Show users what's possible with JuicyDocs
2. **Testing** - Verify that styling works correctly across different content types
